Queue API¶

apirest.queue_serializers.QueueSerializer¶

Queue API is read only, you cannot create Queues using this API at the moment.

strategy: Strategy could have the following values [1: ‘ring-all’, 2: ‘longest-idle-agent’, 3: ‘round-robin’, 4: ‘top-down’, 5: ‘agent-with-least-talk-time’, 6: ‘agent-with-fewest-calls’, 7: ‘sequentially-by-agent-order’, 8: ‘random’, 9: ‘ring-progressively]

Create:

CURL Usage:

curl -u username:password --dump-header - -H "Content-Type:application/json" -X POST --data '{"name": "queue name"}' http://HOSTNAME_IP/rest-api/queue/

Response:

HTTP/1.0 201 CREATED
Date: Fri, 14 Jun 2013 09:52:27 GMT
Server: WSGIServer/0.1 Python/2.7.3
Vary: Accept, Accept-Language, Cookie
Content-Type: application/json; charset=utf-8
Content-Language: en-us
Location: http://HOSTNAME_IP/rest-api/queue/1/
Allow: GET, POST, HEAD, OPTIONS

CURL Usage:
